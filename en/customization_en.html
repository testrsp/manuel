HTML Upload

  1. Click "HTML" below the Smart Editor.
  2. Hold "Ctrl" and "A" to select all unwanted code, then press the "Delete" key.
  3. Paste your HTML code.
  4. Click "Design Mode" to view the design.

Zip Creative Upload

  1. Click "Upload Zip Creative" in the upper right corner of the Smart Editor.
  2. Click "Choose File" and select the Zip file from your computer.
  3. Click "Upload" to view the layout.
    • The Zip file should contain only 1 HTML file and 1 image folder.
    • File size limit: 4M

Set Sender Email

  1. Click "Add Sender Email".
  2. Insert the email address that you want to appear for recipients.
  3. Click "Add Sender Email".
  4. A message will appear: “Verification email has sent, please check and confirm.” Please check the verification email in the mailbox of the email address that you insert.
  5. You can choose this email as your sender email
    • Supports English and Chinese characters and other languages.

Add Reply-to-email

  1. Click "Add Reply-to".
  2. Choose the “Reply-to” email from the drop-down list.
    • Add a new “Reply-to” email: Repeat "Set Sender".

Personalize Email Content

  1. Subject Line

    • Click “Insert a personalized field” near the subject line

  2. Email Body

    1. Click "Personalized Fields".
    2. Insert and then click "Finish".

      Result:
      • Available personalized fields are the template headers. Refer to Upload Contacts.
      • See the table below:
        Column Header Column Header
        A Email Address M Country
        B First Name N Postal Code
        C Middle Name O Sub-Postal Code
        D Last Name P Fax
        E Job Title Q Web URL
        F Company Name R Title
        G Home Phone S Gender
        H Address 1 T Date 1
        I Address 2 U Date 2
        J Address 3 V Birthday
        K City W-AK Custom Field 1-15
        L State AL Status
      • Custom Field 1-15 can be defined by users. See Define Custom Field (Personalized Function).

Define Custom Field (Personalized Function)

  1. Click "My Account".
  2. Click "Define Custom Fields".
  3. Select the field you want to define/rename, e.g., "Custom Field 1".
  4. Fill in the first three spaces (for internal language display).
  5. Click "Modify".
  6. Defined fields will be displayed in the table.
  7. Back to eDM platform, you will find the result of “Define custom field” when insert personalize fields.
