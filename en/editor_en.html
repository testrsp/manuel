What You See Is What You Get Editor

  • Drag and drop any components you want to add

    For example:

    1. Drag the Text field and drop it when a blue bar appears
    2. Input the content
    3. Change the text style and add URL in the Text Editor Bar

Add Pictures

  1. Drag the “Picture” component to drop when a blue bar appears
  2. There are two ways to upload pictures:
    1. Drag and drop pictures directly from PC
      1. Click “Attach File”
      2. Click “Add Files” and choose the pictures from PC
      3. Click “Upload”
      4. Click “Select”
    3. If you have already uploaded pictures to your file manager, you could just “Select” it
      • Picture size Limitation:
        Upload to File Management: 1M
        Drag and Drop: 4M

      • The File Manager can store up to 450M files.

        Even if the File Manager has no storage, users can still add new pictures to a Campaign by “dragging” from the computer.

Grid Layout

  1. Drag and drop the component “Grid Layout”
  2. Set the Grid Layout Style
  3. Click “Component” and select components you would like to combine
  4. Drag and drop to the desired position, release when the orange box appears
  5. Edit the components
  6. Click “Finish”

Add URL (Image and Button)

  1. Click on the component
  2. Click “Content”
  3. Enter the URL
  4. Click “Finish”

Copy, Move, and Delete Component

  1. Copy, move, and delete components using the button in the upper right side of each component
  2. To move the component, click the button below and hold it until a grey bar appears
  3. Drop it to the position you want until a blue bar appears
  4. To delete the component, click the delete button and click “OK” in the pop-up banner
