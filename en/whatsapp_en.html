Basic Procedure

  1. Register a WhatsApp message template.

  2. Select an approved template when creating a campaign.

  3. Upload a file or fill in variables if necessary.

  4. Send the campaign.

Register WhatsApp Template

Interface for register

Steps

  1. Select a sender.

  2. Enter a template name.

  3. (OPTIONAL) Compose header

    1. Select “Multimedia” and upload a file.

      • Please follow the requirements mentioned above.
      • The uploaded file is for reference only. You can upload other files of the same type when creating a campaign.
      • For “Document”, you can also upload image or video when creating a campaign. But there will be no preview in WhatsApp.

    2.   Select “Text” and compose header text.

      • You can add one variable in the header.
      • Please enter an example for each variable.

  1. Compose body

    • The number of variables depends on the length of the body.
    • Please enter an example for each variable.

  2. (OPTIONAL) Compose footer

    • You cannot add any variable in the footer.

  3. (OPTIONAL) Compose buttons

    • You can add buttons up to 10 in total.

    • You can enter up to 25 words in the button text.

    1. Click “Add Quick Reply Button” and fill in the blank.

    2. Click “Add URL Button” and fill in the blanks.

      • You can add up to 2 URL buttons.

    3. Click “Add Call Phone Number Button” and fill in the blanks.

      • You can add up to 1 call phone number button.
      • Please also provide the country code for the phone number.

  1. Click the “Submit” button.

  2. Wait for approval.

Send WhatsApp Campaign

Steps

  1. Select a sender.

  2. Select a template.

  3. If necessary, upload the corresponding type of file.

  4. If necessary, fill in the blanks for each variable.

  5. Provide contact information.

  6. Click “Send”.

Other Functions

  1. Send Test Message

    1. Click "Test".

    2. Fill in the blanks with the phone number.

    3. Click "Send".

  2. Schedule to send the campaign later

    1. Click "Schedule Later".

    2. Click the box next to “Schedule”

    3. Set the time.

    4. Click "Done".

Manage WhatsApp Template

Interface for management

  1. View the date created and the status of each template from the list

  2. Select a template from the list and view template details on the right

  3. Change template name

    1. Change the template name.

    2. Click "Save".

  4. Delete template.

    1. Select a template.

    2. Click "Delete".
