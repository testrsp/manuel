上傳聯絡人

通過上傳 Excel 表格導入聯絡人信息

  1. 點擊巡覽列中的“聯絡人”
  2. 點擊“新增/匯入”
  3. 點擊“聯絡人規範下載”
  4. 根據表格填寫聯絡人信息
  5. 上傳 Excel 文件


    • 請勿修改表格的各項表頭（第一行）
    • “Email address 電郵地址”是重要必填信息（A 列）
    • “Home phone 家庭電話”=移動電話號碼（G 列）
    • 請按照 MM/DD/YYYY 格式輸入“date1 日期1”、 “date2 日期2”和“birthday 生日”（S-V列）
    • “自訂欄位”（1-15）可輸入任何其他信息，如會員 ID、發票號碼、年齡、性別等（W-AK 列），請參閱“個性化郵件內容”和“編輯自訂欄位（個性化功能）”。

更新聯絡人資料

  1. 使用 Excel

    1. “Email address 電郵地址”是用於更新聯絡人資料的重要必填項
    2. 填寫/刪除/修改聯絡人資料
    3. 上傳至 Spread 系統
    4. 請參閱“上傳聯絡人”

    更新聯絡人信息有三個選項：

    1. “更新”：重寫/覆蓋原有信息
    2. “合併”：更新添加的信息
    3. “跳過”：不修改已存在的聯絡人資料，只添加新聯絡人

  2. 使用 Spread 聯絡人管理

    1. 點擊“聯絡人”
    2. 選擇聯絡人列表
    3. 點擊“電郵地址”，選擇指定聯絡人
    4. 填寫/移除/修改聯絡人資料
    5. 點擊“儲存變更”

過濾活躍聯絡人

  1. 點擊“聯絡人”
  2. 點擊聯絡人列表
  3. 聯絡人狀態中選擇“有效的”
  4. 點擊“搜尋”
  5. 點擊“移動到名單”
  6. 彈出提示框，命名新列表
  7. 點擊“移動”，創建新列表成功。
  8. 發送新的郵件時，選擇“有效的”聯 絡人列表。

不再發送名單/啟動

  1. 將人員添加到“不再發送名單”

    1. 點擊“聯絡人”
    2. 點擊“不再發送名單”
    3. 點擊“加入”
    4. 以文字輸入或 Excel 表格的形式，上傳拒收郵件的電郵地址
    5. 點擊“上傳”

  2. 設置“不再發送名單”中的電郵地址為有效

    1. 在同一個頁面，搜索想要設置為有效的電郵地址。
    2. 勾選地址。
    3. 點擊“有效”
